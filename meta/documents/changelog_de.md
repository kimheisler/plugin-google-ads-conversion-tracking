# Release Notes für GoogleAnalytics

## v1.1.4 (2022-25-06)

### Geändert
- Cookie Informationen angepasst und Hinweis zur Rechtslage hinzugefügt. 

## v1.1.3 (2021-11-03)

### Geändert
